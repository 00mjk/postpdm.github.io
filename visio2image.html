Visio to image batch converter

"Visio to image batch converter" is a freeware command line Windows utility for MS Visio documents conversion to PNG images.

Download

MS Visio to image batch converter version, 51 KB zip for Windows, v1.0, release date 11.01.16

Link: visio2image_v1.0.zip

Usage

You need the MS Visio installed on your computer to use the converter.

Command line usage:

visio2image.exe visio_document.ext

Usage example:

visio2image.exe my_draft.vsd

Resulting PNG files will be exported to original Visio document folder as "originalname_page_N.png"
